Lessons Learned



Lessons Learned:

What we have learned creating the tutoring website was adding a more consistent interface to the website as we were still finishing up the website as a whole. Our main goal for the website was to provide the student a way for him to select an appointment of any tutor and be able to have a designated time when trying to reach out to one of the computer science tutors. One thing that went right with the website was the information being displayed for the user to see what computer science classes entailed. The next thing that we want to improve on in this specific type of project is maybe using more options and adding a little more detail maybe to the schedules for the user.

Proposal:

The real world that we want to accomplish is that we want to give students the accessibility online to obtain a tutor when they need help with a specific subject such as Computer Science. The customer/target audience that we are trying to gain is mostly with college students because they are the ones that need help when they are not having very much success in class. The type of value that our website will offer will be a interactive experience with the tutoring website as they can schedule any appointment anytime when they need help. Typical interactive tasks that will be provided for the user will be the user will enter their name inside on of the tutoring slots and click a submit saying they are entered in the system.
